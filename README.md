# datadriven-kundsegmentering-b2b

## Datadriven kundsegmentering för Corporate Banking

## Syfte
Bygga ett beslutsstöd som prioriterar företagskunder baserat på affärspotential.

## Teknik
- PostgreSQL / SQL
- Python (pandas, numpy, scikit-learn)
- Power BI
- Azure (arkitekturperspektiv)

## Metod
1. Datamodellering
2. Kundscore
3. Segmentering
4. Affärsinsikter

## Resultat
- Identifiering av high-value/high-growth-kunder
- Rekommendationer för säljprioritering
